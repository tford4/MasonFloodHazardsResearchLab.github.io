iFLOOD: Chesapeake Bay Flood Forecast System powered by ADCIRC + NAM

Hydrodynamic Model

This system utilizes the Advanced Circulation Model (ADCIRC) Version 52_08_30 in the Two Dimensional Depth Integrated format.

The numerical mesh was developed in the Mason Flood Hazards Research Lab adapted from the FEMA Region 3 High Resolution Mesh containing 0.38 million nodes (381164) with a resolution up to 150 meters inside the Chesapeake Bay.Reference for the publication can be find here: Garzon, J. L. and Ferreira, C. M. (2016)

Atmospheric Model

Meterological forcing using NOAA's North American Mesoscale (NAM) model are used as an input to force water.This system is capable of providing guidance for next three and a half days(3.5) and is automated to run every six hour(6).Forcing from NOAA can be downloaded from here

Computational Resources

The computations are performed in the Mason Flood hazards Research Lab Poseidon Cluster with dedicated 32 cores.

Forecast System

The forecast system was developed using the guidelines from the ADCIRC Storm Surge Guidance (ASGS). The system is run 4 times a day and provides forecasts for 3.5 days in the future.

The North American Mesoscale Forecast System (NAM) is one of the major weather models run by the National Centers for Environmental Prediction (NCEP) for producing weather forecasts. Dozens of weather parameters are available from the NAM grids, from temperature and precipitation to lightning and turbulent kinetic energy.For our forecast system we are only using Atmospheric pressure and wind velocities.


Fig.2 Sea level pressure from Grided NAM model

Outputs from the model run are read in a pythons script to produce Time Series plots against operational model forecasts. Kalpana, an ADCIRC utility is implemented to produce shape files which later are converted to GeoJSON format to use in Google Maps.


Fig.3 Outputs from the Model run

Boundary conditions from the Chesapeake Model are also extracted at the end of simulation to force a locally refined Potomac Flood Forecast Model .Potomac model is also run in realtime and produces flood guidance for the Potomac river.
